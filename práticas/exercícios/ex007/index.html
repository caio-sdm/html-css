Principais formatações

negrito ou destaque (ou negrinho, negresco, negao)

nesta frase, temos um termo em negresco usanto a tag <b> (não semântica)

nessa frase, tenho um termo em destaque usando a tag <strong> (semântica)

itálico ou ênfase

Nesta frase, temos um termo em itálico usando a tag <i> (não semântica)

nesta frase temos um termo em ênfase usando a tag <em> (semântica)

texto marcado (marca texto)

podemos criar um texto marcado usando a tag <mark>

e no outro paragrafo temos outro texto marcado no final

texto gramde e pequeno

estamos criando um texto grande e um texto pequeno nesse paragrafo

texto deletado

podemos marcar um texto como excluido, para indicar que ele nao deve ser considerado

texto inserido

texto inserido para dar enfase e mostrar que foi adicionado depois

tambem pode usar a tag <u> para sublinhados, mas não é semântica

texto sobrescrito

para escrever x20 etc

texto subscrito

para escrever H2O etc
